# BashScript_ConnectToBandit
The purpose of this code is to connect to database server that was made on the local machine, where passwords for each level of the bandit game are stored. After getting the password for desired level using Bash and SQLite, code automatically connects to bandit server-level.

All personal data like IP address was removed from the code. To connect to the local database, users of this code need to input their own local machine name and IP address
